## 🚀 What is TheAIgentsCompany?

**TheAIgentsCompany** is a **multi-agent orchestration platform** where specialized AI agents collaborate to build, manage, and evolve software projects from idea to delivery — entirely through autonomous agent workflows.

### 🏗️ The Architecture

```
┌─────────────┐
│  👑 CEO     │  ← Orchestrator — sole user interface
└──────┬──────┘
       │
       ├──→ 🔧 Engineering Agent   (Dev)
       ├──→ 🧠 AI/ML Agent         (AI/ML)
       ├──→ 🔬 Research Agent      (Research)
       ├──→ 📊 Product Agent       (Product)
       ├──→ ⚙️ Operations Agent    (Ops)
       ├──→ ☁️ Infrastructure      (Infra)
       ├──→ 🎨 Front-End Designer  (FE)
       ├──→ 🔧 Git & GitHub        (Git)
       └──→ 📚 Documentation       (Docs)
```

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| **🎯 Centralized Orchestration** | CEO Agent is the sole interface — no peer-to-peer agent communication |
| **🧩 10 Specialized Agents** | Each agent has a precise domain: Dev, Research, Infra, AI/ML, Product, Ops, Design, Git, Docs |
| **📋 Project Lifecycle** | 7 phases from intake to delivery — fully structured workflow |
| **📊 Interactive Dashboard** | Real-time agent catalog and system status visualization |
| **📚 Mintlify Documentation** | Clean, searchable documentation site (coming soon) |
| **🐳 Docker-First** | Containerized deployment for all platform components |

### 🧠 Core Principles

1. **CEO is the only interface** — No agent communicates directly with the user
2. **No peer-to-peer** — All communication flows through the CEO
3. **Core agents only** — Projects use existing core agents, never spawn new ones
4. **Two-repo model** — Core repo (platform) + one repo per project
